# Статус импорта сессий

## Исправлено

✅ **Формат session_data исправлен:**
- `android_version` теперь число (int), а не строка
- `device_id` добавляется в корень session_data
- Убраны None значения из device_settings
- Сессия устанавливается без ошибок формата

## Проблема

❌ **Сессии из файла устарели (2022-2023 год):**
- Сессии не могут подключиться к Instagram
- Ошибка: "Network is unreachable" или "Connection timeout"
- Instagram требует свежие сессии (экспортированные недавно)

## Решение

1. **Экспортировать свежие сессии из браузера:**
   - Авторизоваться в браузере через прокси
   - Экспортировать cookies сразу после авторизации
   - Использовать инструкцию в `HOW_TO_EXPORT_COOKIES.md`

2. **Импортировать через API:**
   ```
   POST /api/accounts/{account_id}/import-session
   {
     "session_data": {
       "cookies": {...},
       "device_id": "...",
       "user_agent": "..."
     }
   }
   ```

## Важно

- Сессии должны быть свежими (экспортированы сразу после авторизации)
- Использовать тот же прокси, через который авторизовались в браузере
- Сессии из 2022-2023 года не работают - нужны свежие
